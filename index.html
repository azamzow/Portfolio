Code Chat

During my Fall semester in 2017, I worked with a 9-man team to design and build any project we desired. Some of my team mates had been working together previously and had tried sharing code between each other, However they found no optimal way.

We decided to build a chatting application that could run and compile code immediately. I was the assistant programmer for all JavaScript modules within back-end. Our whole system utilizes websockets between the webapp, the chat server, and the compiling server, making it as network performant as possible.

It allows managers, and the part-time students who are the employees. (Hence the Working Kats, because SHSU's mascot is a bearkat).
