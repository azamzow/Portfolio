Build-A-Bar

Build-A-Bar is a way to design and personalize your own emergancy lightbar. The user begins with two options, single or dual color. The dual option allows reversal of the LED's and specific color cordinations such as "Dual Flood" or "Amber Arrow". Color cordination differes within each product. The user can then 'build a bar'. Build-A-Bar is unique because no other Emergancy Lighting company provides customization like this.

This project was built using Angular, JavaScript, and HTML with SASS and CSS. Each bar's information is stored using an object. Once an object is created it gets pushed in an array. When the bar is complete, the array is sorted using bubble sort and is sent to the manufacturing department. The user also has the option to download a PDF of their built bar. This ensures that each order is accurate.
